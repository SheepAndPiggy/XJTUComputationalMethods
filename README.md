## 📖 简介

**XJTU计算方法课程C++代码库** 是一个西安交通大学研究生课程——计算方法的算法实现库，为作者练习C++编程以及学习计算方法所写，涉及的算法均在《数值分析》中详细讲解。

这个代码库希望为学习数值计算和学习计算方法的同学们提供一个参考，所有代码可自取。
